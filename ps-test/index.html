Perceptual Speed Test (ε and ¥)

In this test, you will see strings with various symbols. Your task is to quickly decide whether both target symbols (ε and ¥) are present in the string.

When both symbols are present, please press the J key. If only one or neither of the symbols is present, please press N. You will receive one point for each correct input and a minus point for each incorrect one.

Should you accidentally press either key twice in a row, you can correct your last input using the "Back" button. However, only the most recently submitted answer can be changed.

The search test begins with a 30-second test phase, during which you can familiarize yourself with the procedure. You can then start the actual test by clicking the "Start Final Test" button.

The main test follows the same procedure as the test phase. However, it lasts 120 seconds.

If you have read the instructions, please enter your desired user ID and then start the test phase.
